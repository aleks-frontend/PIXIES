<p><b>Configuración de Email</b> (<a href="%help_file%1084.html" target="_blank">Ayuda</a>)</p>
<blockquote>
	<p>Proporcione la información debajo para que este programa se pueda poner en contacto con usted. Actualmente el programa se puede configurar para que le envié un Email cuando alguien añade un URL nuevo (vea Reglas de Filtro).</p>
	<p>Después de fijar la información de email, <a href="%admin_url%&amp;Action=PS&amp;subaction=TestMail">envíe un mensaje de prueba</a> para confirmar la configuración.</p>
	<p>Use esto <a href="http://www.xav.com/mx_lookup.pl">Herramienta SMTP de búsqueda del servidor</a> si usted necesita ayuda para determinar su servidor de mail.</p>

	</tr>
	<tr valign="top" class="fdtan">
		<td align="right"><b>Dirección de Email:</b></td>
		<td>
			<input name="admin notify: email address" /><br />
			<font size="-2">ex: admin@xav.com</font>
		</td>
	</tr>
	<tr valign="top" class="fdtan">
		<td align="right"><b>Servidor SMTP:</b></td>
		<td>
			<input name="admin notify: smtp server" /> opcional<br />
			<font size="-2">ex: mail.xav.com</font>
		</td>
	</tr>
	<tr valign="top" class="fdtan">
		<td align="right"><b>Programa de enviar mail:</b></td>
		<td>
			<select name="admin notify: sendmail program">%sendmail_options%</select> opcional<br />
			<font size="-2">ex: /usr/sbin/sendmail -t</font>
		</td>
	</tr>
	</table>
	<p>Para enviar correo, esta escritura necesita un servidor del SMTP o un programa de sendmail. Si se definen ambos, el programa de sendmail será utilizado.</p>
